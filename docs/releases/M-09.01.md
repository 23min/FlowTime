# M-09.01 - Engine Semantics Layer (Contract Hardening)

**Status:** Complete  
**Release Date:** 2026-01-21  
**Branch:** `milestone/m-09.01`

## Overview

M-09.01 establishes a documented semantics catalog and hardens API contracts around series metadata, including edge series. This milestone focuses on pass-through metadata fidelity and test coverage, plus stabilizes CLI/API parity testing.

### Highlights
- **Semantics catalog:** Added a consolidated reference for series units, origin, aggregation, and warning families.
- **Edge series metadata:** Edge series now carry `seriesMetadata` alongside node series metadata.
- **CLI parity stability:** CLI parity tests now invoke the built CLI DLL with timeouts to avoid hangs.

## Contract Updates

- **`time-travel-state` schema**
  - `edges[].seriesMetadata` added for per-series origin/aggregation metadata.
