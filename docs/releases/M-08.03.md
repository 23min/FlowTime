# M-08.03 - MCP Data Intake and Profile Fitting

**Status:** Complete  
**Release Date:** 2026-01-19  
**Branch:** `milestone/m-08.03`

## Overview

M-08.03 adds data intake and profile fitting for MCP-based modeling. FlowTime.Sim now ingests pre-aggregated series, summarizes inputs, fits profiles or PMFs, previews results, and applies them to draft templates with provenance metadata. The MCP server surfaces these endpoints as tools for chat-driven modeling.

### Highlights
- **Series intake + summaries:** ingest CSV or pasted tables, validate, and summarize with optional expert diagnostics.
- **Profile fitting:** fit time-pattern profiles or PMFs from series, samples, or summary stats.
- **Draft mapping:** apply profiles/PMFs directly to draft nodes with provenance metadata.
- **MCP tool surface:** new tools for ingest, summarize, fit, preview, and mapping.

## API Changes

- **New Sim endpoints:**
  - `POST /api/v1/series/ingest`
  - `POST /api/v1/series/summarize`
  - `POST /api/v1/profiles/fit`
  - `POST /api/v1/profiles/preview`
  - `POST /api/v1/drafts/map-profile`

## Tooling Updates

- MCP tools: `ingest_series`, `summarize_series`, `fit_profile`, `preview_profile`, `map_series_to_inputs`.
- Responses support `detailLevel` (`basic` or `expert`) for guidance vs diagnostics.
