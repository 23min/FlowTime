# M-08.02 - MCP Modeling Draft Workflow

**Status:** Complete  
**Release Date:** 2026-01-19  
**Branch:** `milestone/m-08.02`

## Overview

M-08.02 adds a draft authoring loop for MCP-based modeling and extends FlowTime.Sim with draft-specific endpoints so a single Sim instance can validate, generate, and run drafts without touching approved templates.

### Highlights
- **Draft authoring tools:** MCP server supports create, get, patch, diff, validate, generate, and run for draft templates.
- **Single Sim instance:** New draft endpoints accept inline YAML or draftId sources under `templates-draft/`.
- **Safety guardrails:** Drafts stay isolated from `templates/` and include content hash protection.
- **Run integration:** Draft runs produce standard run bundles and can be imported into the Engine for inspection.

## API Changes

- **New Sim endpoints:**
  - `POST /api/v1/drafts/validate`
  - `POST /api/v1/drafts/generate`
  - `POST /api/v1/drafts/run`

## Tooling Updates

- MCP draft tools: `create_draft`, `get_draft`, `apply_draft_patch`, `diff_draft`, `validate_draft`, `generate_model`, `run_draft`.
- Draft storage defaults to `templates-draft/` with content hashes to protect against concurrent edits.
