# M-07.01 Release Notes

**Release Date:** 2026-01-18  
**Milestone:** M-07.01 - Edge Time Bin Foundations  
**Status:** ✅ Complete  

---

## 🎯 Milestone Goal

Introduce edge time-bin metrics across schemas, artifacts, simulation, and API so edges become first-class flow facts without requiring template changes.

## ✅ What's New

### Edge Time-Bin Foundations
- **Schema/Contracts:** Optional `edges` added to time-travel state responses, including `flowVolume` and optional `byClass`.
- **Artifacts:** Edge series emitted into run artifacts and indexed alongside node series.
- **Simulation:** Edge flow totals (and per-class breakdowns when classes exist) emitted for synthetic runs.
- **API:** `/state` and `/state_window` surface edge data when present.

### Reliability Improvements
- **CLI integration test stability:** Read CLI stdout/stderr concurrently with a timeout to avoid deadlocks.
- **Perf gating:** PMF mixed workload perf test deferred; recorded in perf log.
