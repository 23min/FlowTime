# Release M-06.02 — Model Compiler (Compile-to-DAG Extraction)

**Release Date:** 2026-01-18  
**Type:** Milestone delivery  
**Canonical Run:** TBD

## Overview

M-06.02 extracts compile-to-DAG expansion into a core `ModelCompiler` so every entry point evaluates a compiled model consistently. Queue depth and retry echo synthesis now live in the compiler, with parity tests confirming outputs match previous artifact writer behavior.

## Key Changes

1. **Core model compiler**  
   Introduced `ModelCompiler` to expand queue depth and retry echo series during compile-to-DAG.  
   (`src/FlowTime.Core/Compiler/ModelCompiler.cs`)
2. **Shared evaluation pipeline**  
   Artifact writer, generator evaluation, and invariant analysis now compile before evaluation.  
   (`src/FlowTime.Core/Artifacts/RunArtifactWriter.cs`, `src/FlowTime.Generator/Orchestration/RunOrchestrationService.cs`, `src/FlowTime.Sim.Core/Analysis/TemplateInvariantAnalyzer.cs`)
3. **Parity + analyzer coverage**  
   Added tests ensuring compiled models match prior outputs across entry points and suppress false invariant warnings.  
   (`tests/FlowTime.Core.Tests/Compiler/ModelCompilerParityTests.cs`, `tests/FlowTime.Generator.Tests/RunOrchestrationServiceTests.cs`, `tests/FlowTime.Sim.Tests/TemplateInvariantAnalyzerTests.cs`)
