# UI Foundations & Runner

**Status:** Retrospective completed epic grouping.

This folder groups the early pre-charter UI milestones that established the first runnable FlowTime UI, introduced the template-driven simulation runner, and replaced mock-only flows with artifact-first API integration.

## Themes

- Bootstrap the first Blazor-based FlowTime UI surface.
- Introduce template and catalog driven simulation workflows.
- Replace mock service flows with real API-backed artifact-first execution.

## Milestones

- `UI-M-00.00` — initial UI viewer, charting, structural graph, and simulation toggle.
- `UI-M-01.00` — template-based simulation runner with schema-driven forms and catalog integration.
- `UI-M-02.00` — real API integration, artifact-first result loading, and mode-toggle/UX hardening.

## Notes

- These milestones predate the current numbered epic workflow.
- The grouping is retrospective and is intended to localize archived milestone history under an epic-local structure.
