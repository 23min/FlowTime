# PMF Modeling

**Status:** Retrospective completed epic grouping.

This folder groups the first probabilistic modeling extension for FlowTime: discrete PMF inputs converted into deterministic expected-value series that remain compatible with the engine's core execution model.

## Themes

- Introduce uncertainty modeling without abandoning deterministic evaluation.
- Make PMF nodes interoperable with the expression language and time-grid pipeline.
- Establish the first modeling path for probabilistic inputs ahead of later analytical work.
