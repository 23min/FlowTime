# Milestones Directory

`work/milestones/` is a compatibility stub, not the source of truth for current work.

Use epic-local milestone files instead:

- milestone specs live in the owning epic folder under `work/epics/<epic>/`
- milestone logs live beside the spec as `*-log.md`

Rules:

- do not create new milestone specs in `work/milestones/`
- do not use `work/milestones/tracking/` for active work
- keep this directory empty except for this README unless a legacy artifact is genuinely unmapped and awaiting a human placement decision

This directory still exists because some older docs and framework/tooling paths have not been fully migrated yet.
