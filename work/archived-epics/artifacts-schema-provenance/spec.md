# Artifacts, Schema & Provenance

**Status:** Retrospective completed epic grouping.

This folder groups the charter-foundation milestones that turned FlowTime from an ephemeral execution surface into a persistent artifacts platform with export flows, registry APIs, schema evolution, and provenance-aware querying.

## Themes

- Establish export formats and artifact storage as first-class platform capabilities.
- Build and enhance the persistent artifacts registry for charter-era workflows.
- Complete schema evolution and provenance support across engine-facing contracts.

## Milestones

- `M-02.06` — export system implementation and multi-format artifact generation.
- `M-02.07` — file-based persistent artifacts registry.
- `M-02.08` — enhanced registry APIs and shared-artifact service support.
- `M-02.09` — schema evolution, documentation alignment, and provenance support.
- `M-02.09.01` — TDD setup and migration test-plan companion for the M-02.09 thread.
- `M-02.10` — provenance query support in API and CLI surfaces.
