# Skill: milestone-wrap

Purpose: complete a milestone without merging to main.

Use when:
- All acceptance criteria are met and tests pass.

Process:
1) Mark milestone status ✅ Complete.
2) Update tracking doc with final test results and summary.
3) Update docs listed in docs/development/milestone-session-guide.md.
4) Do not merge to main; stay on the branch.

Outputs:
- Milestone closed and documented.
- Docs updated and consistent.
